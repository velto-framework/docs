Velto Tags

Velto v0.4 introduces a significantly expanded tag set. Tags are semantic, Velto‑native, and map to HTML elements at runtime. This section documents every built‑in tag and its default mapping.

<velto>

The root element of every Velto document.

  • Required: YES
  • Children: any Velto element
  • Maps to: <div>

<content>

Main content container.

  • Maps to: <div>

Unknown Tags

Unknown tags are allowed. The reference runtime maps them to <div> and logs a warning. Future versions of Velto may support custom tag registration.
